# 🧪 JS State & Routing Playground

This repo is my personal sandbox for exploring different **JavaScript/TypeScript state management libraries** and other modern tools I want to get more comfortable with.

## What’s inside?

- Experiments with popular **state management solutions** (Zustand, Jotai, Redux Toolkit, etc.)
- Demos with **TanStack Form** and **TanStack Router**
- Styling with **Ant Design** (antd)
- Small example apps & snippets to compare patterns, ergonomics, and performance
- Notes and takeaways from trying out different approaches
